# 模板改进建议：增加"跨市场指标补充"pack

> 提出日期：2026-05-05
> 提出人 / agent：Claude（v0.2 试跑撰写 SMIC HK 报告时发现）
> 状态：pending

## 出现在哪份报告

- 路径：`research/stocks/hk/00981-HK-smic-20260505.md`
- 章节：§5.3（监管 / 地缘 / 合规）+ §6.1（估值表）+ §7.3-7.4（强化/削弱事件）

## 建议改哪个文件 / 哪一节

- 文件：`research/templates/references/industry-metrics.md`
- 章节：新增独立小节"跨市场上市公司补充指标"（与行业 packs 平级）

## 问题描述

写 HK 报告时，多个跨市场专属指标无处安放，被零散塞进了 §5.3 / §6.1 / §7：
- AH 溢价 / 折价（历史序列、当前值、警戒区间）
- 南向资金持仓变动
- 港股通名单变动
- ADR 与本地股价差
- 国际指数权重（MSCI / FTSE）
- 双重上市的审计与合规风险（PCAOB 等）

如果每次写中概股、AH 双股、港美双股都要"现造轮子"，模板效率低且容易遗漏。

## 建议改法

在 `references/industry-metrics.md` 增加独立小节（与 5 个行业 packs 平级）：

```markdown
## 跨市场上市公司补充指标 (cross-market)

**触发条件**：报告 frontmatter 的 `markets` 字段含 ≥2 个市场（如 [cn, hk] 或 [hk, us]）。

### 估值补充
- 当前 AH 溢价 / 折价 + 历史分位
- 主市场 / 次市场流动性对比（日均成交额、做市深度）
- ADR 折溢价（如适用）

### 风险补充
- 南向 / 北向资金持仓变动
- 港股通 / 沪深港通名单变动
- 国际指数（MSCI / FTSE 等）权重变动
- 双重上市审计风险（如 PCAOB）
- 多地货币 / 联系汇率风险

### Sanity 补充
- AH 折价是否在历史合理区间（>50% 通常需要解释）
- 主次市场估值倍数差异是否被流动性差异充分解释
```

主 spec 在 §1 / §5 / §6 中相应位置加一句"跨市场标的请同时引用 industry-metrics.md → 跨市场补充指标"。

## 优先级

**中**——单一市场标的不受影响；但 AH 双股、中概股、港美双股报告都会用到
