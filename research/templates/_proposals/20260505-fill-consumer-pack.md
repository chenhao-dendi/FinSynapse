## 4. 建议改法

参照半导体 pack 的结构填实，骨架：

```markdown
## 消费 (consumer)

### 触发条件
报告 frontmatter `industry: consumer`。

### 子分类
- 必选消费（食品饮料、日用品、家庭清洁）
- 可选消费（服装、潮玩、化妆品、奢侈品）
- 服务消费（餐饮、教育、旅游、娱乐）
- 耐用消费（家电、家具、汽车）
- IP / 版权类（玩具、动漫、游戏周边） ← 子赛道独立补充包，见另一 proposal

### 业务画像补充（§1.4 末尾）
| 指标 | 描述 / 口径 |
|------|------------|
| 子赛道 | 见上 |
| 直营 vs 加盟比例 | 直营 / 加盟 / 经销 三种模式占比 |
| 中国门店数 / 海外门店数 | 年末门店数 + 净开店趋势 |
| 主要品牌矩阵 | Top 3-5 品牌与对应营收占比 |
| 渠道结构 | 直营 / 电商 / 经销 / KA 商超等 |
| 客户画像 | 年龄、性别、地域、收入分布 |
| 客户集中度 | C 端散客（极分散）vs B 端经销（中等集中） |

### 财务体检补充（§2.5 末尾）
| 指标 | 描述 | 优先级 |
|------|------|:------:|
| 同店销售增长 SSSG | 已运营满 1 年门店的销售增速 | 必填（零售） |
| 单店年化营收 | 中国 vs 海外分项 | 推荐 |
| 坪效 / 人效 | 万元/平米/年 / 万元/人/年 | 推荐 |
| 库存周转天数 | (库存 / 营业成本) × 365 | 必填 |
| 应收账款周转 | C 端通常极快，B 端波动大 | 推荐 |
| 销售费用率 | 含广告、推广、门店运营 | 必填 |
| 复购率 / 会员粘性 | 公司主动披露较少，能拿到则非常有价值 | 选填 |
| NPS / 客户满意度 | 通常不公开 | 选填 |
| ARPU / 客单价 | 单次消费金额 | 推荐 |

### 护城河 / 竞争补充（§3.2 对手对比表）
建议同业对比表至少含：
- 主要品牌矩阵
- 直营比例
- 单店模型差异（高线 vs 下沉）
- 全球化进度

### 风险特别提示（§5.3 监管/地缘 末尾）
| 风险类别 | 关注点 |
|---------|-------|
| 行业监管 | 食品安全 / 消费品质量 / 玩具安全 / 广告法 |
| 关税与贸易 | 跨境电商关税 / 海外市场进口政策 |
| 知识产权 | 山寨、品牌侵权、商标海外注册 |
| 消费周期 | 可选消费在宏观下行时承压最大 |
| 数据合规 | 会员系统、跨境数据传输 |

### Sanity Checks 补充
- [ ] 营收增速与门店净开店速度方向一致
- [ ] 毛利率与渠道结构匹配（直营高、加盟中、经销低）
- [ ] 同店增长与新店扩张共同贡献营收增速（不能完全靠开店）
- [ ] 库存周转与营收增速方向一致（库存累积 + 营收滞涨 = 周期顶部信号）
- [ ] 销售费用率与新市场扩张节奏匹配

### 估值参考区间
| 子赛道 | 通常 PE | 通常 PS | 周期特征 |
|--------|--------|--------|---------|
| 必选消费（成熟） | 18-30 | 1.5-3 | 弱周期 |
| 可选消费（成熟） | 12-25 | 1-3 | 强周期 |
| 高成长 / 出海期 | 30-60+ | 3-10 | 叙事溢价 |
| 餐饮 / 服务 | 15-30 | 1-3 | 中周期 |
| IP / 版权类 | 见 IP 子包 | 见 IP 子包 | — |
```
