## 3. 问题描述

撰写 Pop Mart 报告时发现，通用消费 pack 的指标（同店、库存、复购）**不足以覆盖 IP 类公司独有的关键维度**：
- IP 矩阵集中度（Top 5 IP 营收占比）
- 自有 IP vs 授权 IP vs 艺术家分成 三类成本结构
- 单 IP 生命周期与营收波动
- 海外破圈方法论（明星带动 / 本土 IP / 文化适配）
- 二手 / 收藏价值（IP 价值不在销售时结束）

这些维度不属于"通用消费"，但又显著影响 IP 类公司（玩具、动漫、游戏周边、潮玩、Fan 经济）的研究。

## 4. 建议改法

在 `industry-metrics.md` 的"消费 pack"下新增子包：

```markdown
### 子赛道扩展：IP / 版权类 (consumer-ip)

**触发条件**：消费类公司中，IP 是核心驱动力（如潮玩、动漫周边、玩具、游戏 IP 衍生）。报告 frontmatter 可加 `industry_subtype: ip` 标记。

#### 业务画像补充
| 指标 | 描述 |
|------|------|
| IP 矩阵 Top N | Top 3 / Top 5 / Top 10 IP 的营收占比与生命周期阶段 |
| 自有 vs 授权 vs 分成 | 三种 IP 来源占比 + 各自毛利率差异 |
| 头部 IP 集中度 | Top 1 IP 占比 > 30% 通常意味着集中度风险 |
| IP 联名 / 跨界 | 与其他 IP / 品牌联动的合作数量与营收贡献 |

#### 财务体检补充
| 指标 | 描述 |
|------|------|
| 自有 IP 毛利率 vs 授权 IP 毛利率 | 通常自有 > 授权 > 分成 |
| IP 资本化与摊销 | 部分公司将 IP 收购费用资本化分摊 |
| 艺术家 / IP 持有人分成结构 | 公开信息少，但是关键风险点 |

#### 护城河补充
| 维度 | 关注点 |
|------|------|
| IP 矩阵深度 | 单一爆款 vs 多 IP 平台 |
| IP 生命周期管理 | 衰退期续命、复古回归、跨媒介延伸 |
| 二手 / 收藏价值 | 二手价格指数（如有公开追踪） |
| 用户社区运营 | APP 月活、UGC 内容量、二手交易频次 |

#### 风险特别提示
| 风险 | 关注点 |
|------|------|
| 单 IP 依赖 | Top 1 IP 占比过高 |
| 授权续约 | 重要授权 IP 的合作期限与续约风险 |
| IP 周期 | 历史平均潮玩 IP 生命周期 3-5 年 |
| 文化适配（出海） | 中国 IP 在海外的文化共鸣度 |
| 山寨与盗版 | 维权成本与品牌损耗 |

#### Sanity Checks 补充
- [ ] 增量营收的 IP 来源结构是否健康（不是单 IP 拉动 70%+）
- [ ] 授权 IP 的合同到期时间与公司战略一致性
- [ ] 头部 IP 的月度销售环比是否进入衰退（出现 3 个月连续下滑通常是周期顶部信号）

#### 估值参考区间
| 类型 | 通常 PE | 通常 PS | 备注 |
|------|--------|--------|------|
| 单 IP 公司（如个别动漫/玩具） | 15-25 | 2-4 | 周期波动大 |
| 多 IP 平台（孩之宝、万代）| 12-22 | 1.5-3 | 成熟 |
| 高成长 IP 平台（出海期） | 30-60+ | 5-15 | 叙事溢价显著（如 Pop Mart） |
| 顶级 IP 综合体（迪士尼） | 20-30 | 2-3 | 多元业务 |
```

## 7. 风险与权衡

- 若引入 `industry_subtype` 字段：会让 frontmatter 复杂化，但收益是子包清晰定位
- 若不引入：agent 需自行判断公司是否为 IP 类，可能漏读子包

## 8. 替代方案

- **方案 A**（推荐）：在消费 pack 下做"子包"层级，frontmatter 加可选 `industry_subtype`
- **方案 B**：单独建一个 `industry: ip` 顶级行业（与 consumer 平级）—— 但破坏现有行业划分
- **方案 C**：作为消费 pack 的"附录"段落，不做正式子包 —— 命名混乱
