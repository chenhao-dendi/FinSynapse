> 适用：所有遵循 `templates/company-analysis.md` 的研究报告。报告 frontmatter 的 `industry` 字段决定引用哪个 pack。
> 版本：0.2（半导体 pack 已填充；其余 pack 按需求出现时再补）

通用模板（`company-analysis.md`）覆盖**任何公司都适用的财务/治理/估值维度**。但银行的 NIM、SaaS 的 ARR、半导体的产能利用率，无法用通用维度替代。本文件按行业列出**额外应在 1.4 / 2.x 章节补充的指标清单**。

## 半导体 (semiconductor)

> 状态：v0.2 已填充（基于 SMIC 试跑反馈沉淀）。覆盖晶圆代工 / IDM / Fabless 通用维度，部分指标对设备/材料/EDA 子赛道仍需扩展。

### 触发条件
报告 frontmatter `industry: semiconductor`。

### 子分类（§1.1 商业模式应归一到其中）
- **晶圆代工 (Foundry)**：仅做制造，无设计（如 SMIC、TSMC、UMC、华虹）
- **IDM**：设计 + 制造一体（如 Intel、Samsung、TI、ADI）
- **Fabless**：仅做设计，外包制造（如 NVIDIA、Qualcomm、海思、寒武纪）
- **设备**：光刻 / 刻蚀 / 薄膜 / 检测设备（如 ASML、Lam、AMAT、KLA、中微、北方华创、SMEE）
- **材料**：硅片 / 光刻胶 / 电子特气 / 靶材（如沪硅产业、JSR、TOK、华特气体）
- **EDA / IP**：芯片设计软件与 IP 核（如 Synopsys、Cadence、ARM、华大九天）
- **封测 (OSAT)**：先进封装、测试（如日月光、长电科技、通富微电）

### 业务画像补充（写在主模板 §1.4 末尾"行业指标补充"位置）

| 指标 | 描述 / 口径 |
|------|------------|
| **子赛道** | 见上"子分类"，必填 |
| **制程节点分布** | 按节点（如 0.18μm / 65nm / 28nm / 14nm / 7nm / 5nm / 3nm）的营收占比 |
| **工艺类型** | 逻辑 / 存储（DRAM/NAND） / 模拟 / 功率 / RF / 传感 / 特殊工艺 |
| **客户集中度** | Top 5 / Top 10 占比；Top 1 客户名（如能识别） |
| **客户行业分布** | 智能手机 / PC / AI / 汽车 / 工业 / 物联网 / 消费 / 通信 |
| **地域分布** | 中国大陆 / 北美 / 欧洲 / 亚太其他（按收入） |

### 财务体检补充（写在主模板 §2.5 末尾"行业指标补充"位置）

| 指标 | 描述 / 口径 | 优先级 |
|------|------------|:------:|
| **月产能（8 寸等效）** | wafer/月，需折算到 200mm 等效；12 寸需 ×2.25 折算 | 必填（Foundry/IDM） |
| **产能利用率** | 季度 / 年度均值，% | 必填（Foundry/IDM） |
| **Capex / 营收** | 资本开支强度，反映周期位置 | 必填 |
| **折旧 / 营收** | 高资本投入回收节奏 | 推荐 |
| **R&D / 营收** | 反映技术路线投入；先进制程 / 设备 / EDA 必看 | 推荐 |
| **设备保有量与购买计划** | 关键卡脖子设备（如 EUV/DUV/HBM 设备） | 推荐（Foundry） |
| **良率** | 公司主动披露较少，能拿到则非常有价值 | 选填 |
| **ASP（晶圆/芯片均价）** | 公司不强制披露，可用 营收 / 出货量 估算 | 选填 |
| **库存周转天数** | 周期判断（>120 天通常意味着需求弱） | 推荐 |
| **DOI（库存指数）** | 同业横向比较 | 选填 |

### 护城河 / 竞争补充（写在主模板 §3.2 对手对比表中）

建议同业对比表至少含：
- **全球市场份额**（依子赛道）
- **先进制程能力**（最新已量产节点）
- **毛利率 / Capex 强度** 横向对比
- **研发费用率**

### 风险特别提示（写在主模板 §5.3 监管/地缘/合规 末尾）

| 风险类别 | 关注点 |
|---------|-------|
| **出口管制** | 美国 EAR/Entity List / 商务部 BIS 新规 / 荷兰对 ASML EUV+DUV 限制 / 日本特定材料管制 |
| **关键设备/材料供应** | EUV / DUV 浸没式 / HBM / 高端光刻胶 / 12 寸大硅片 / EDA / IP |
| **客户端制裁风险** | 主要客户被列入实体清单 / 产品端被禁止销往特定国家 |
| **地缘热点** | 台海 / 中美 / 朝鲜半岛 |
| **库存周期** | 全球半导体库存指数（DOI） / 终端需求（手机 / PC / 汽车 / AI 服务器） |
| **价格战** | 成熟制程产能过剩时段，单价快速下行 |

### Sanity Checks 补充（写入 §6.x 估值或 §7 结论时核对）

- [ ] **产能利用率与营收增速方向一致**：利用率 ↑ 但营收 ↓ 需解释（通常意味着 ASP 大跌）
- [ ] **Capex 强度与同业对比**：偏离 ±50% 需解释（极高 = 周期上行 / 战略扩张；极低 = 周期下行 / 资金紧张）
- [ ] **毛利率与产能利用率匹配**：85%+ 利用率下毛利率仍 <15% 通常意味着产品组合或定价问题
- [ ] **库存周转天数与终端需求方向一致**：库存累积 + 营收滞涨 是周期顶部信号
- [ ] **R&D 费用率**：先进制程公司 <8% 通常意味着技术路线减速

### 估值横向定位（必须构建 peer set）

半导体子赛道差异极大，不能把无来源的"合理区间"写成硬规则。§6.1 做估值对比时必须先构建 peer set：

| 步骤 | 要求 |
|------|------|
| 1. 选样本 | 同一子赛道优先，如 Foundry 只和 Foundry 比；跨子赛道比较必须说明原因 |
| 2. 统一口径 | 同一交易日市值、同一币种、同一财年或 TTM/LTM 指标 |
| 3. 给分布 | 至少列 peer median、25/75 分位，样本少于 5 家时标注"样本不足" |
| 4. 标来源 | 每个 peer 的价格、财务指标、as_of、来源必须可反查 |
| 5. 解释偏离 | 与 peer median 差异 >50% 时，必须解释是周期错位、盈利质量、国家战略叙事、AI 叙事、流动性差异还是口径问题 |

如需要使用 benchmark 区间，必须在表格题注写明来源、样本范围、统计日期和口径；无来源区间不得作为 hard gate。

### 关键监控数据源（写入 §5.4 关键事件日历或 §7 跟踪信号）

| 数据 | 频率 | 来源 |
|------|------|------|
| 公司季报（营收 / 毛利率 / 产能利用率） | 季度 | 公司 IR / 交易所 |
| WSTS 全球半导体销售数据 | 月 | wsts.org |
| TrendForce / SEMI / Gartner 行业报告 | 季 / 年 | 三方付费 |
| SOX 半导体指数 / 费城半导体指数 | 日 | 彭博 / Wind |
| 美国 BIS 出口管制新规 | 不定期 | bis.doc.gov |
| 主要客户出货数据 | 季 / 月 | 终端公司财报（如 Apple / NVIDIA / 华为） |
