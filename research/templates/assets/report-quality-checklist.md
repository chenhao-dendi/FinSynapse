# 报告质量自检 Checklist (v0.2)

> 适用：所有遵循 `templates/company-analysis.md` 的研究报告。
> 用法：报告写完后，agent 必须逐项勾选；任何未通过项必须在报告 `## Meta` 段中说明原因。
> 版本：0.2（11 条；新增 A6 评级矩阵自检 + A4 激励数据例外说明）

---

## A. 完成度（6 条）

- [ ] **A1 frontmatter 完整**：`title / as_of / tickers / markets / industry / template_version / depth / source_level / confidence / not_investment_advice / sources` 全部填写，未填的字段不能省略而是标注 `N/A` 并解释
- [ ] **A2 TL;DR 独立可读**：≤200 字；只读 TL;DR 也能知道公司业务、当下叙事、多空核心、研究结论
- [ ] **A3 关键数字有审计线索**：财务、估值、市场份额数字均带 `as_of` 与来源（规则见 `references/data-source-policy.md`）
- [ ] **A4 N/A 有解释**：所有标注 N/A 的章节，在 `## Meta → 不适用章节` 中说明原因
  - **例外**：§4.2 管理层激励数据公开披露不充分时，标"未公开披露"即可，无需在 Meta 列为 N/A 章节（详见主 spec §5.5）
- [ ] **A5 结论结构完整**：第 7 章包含 thesis（一句话）+ 强化事件（≥3 条）+ 削弱事件（≥3 条）+ 证伪事件（≥1 条）+ 重审日期
- [ ] **A6 §7.0 维度评级矩阵已填**：6 维（业务地位 / 财务健康 / 护城河 / 治理与资本配置 / 成长驱动 / 风险抵御）每行 ★1-★5 + 一句话理由（≤30 字）；矩阵综合判断应与 7.1 单一标签一致

## B. Sanity Checks（5 条）

- [ ] **B1 财务关系合理**：通常应满足 **毛利率 ≥ 经营利润率 ≥ 净利率**。如不成立（一次性损益、特殊会计处理），必须在正文中解释，不能让读者自行推测
- [ ] **B2 估值倍数在行业合理区间**：PE/PS/EV-EBITDA 与同业差异 > 50% 时，必须在 6.1 解释（是叙事溢价、盈利质量差异、还是周期错位）
- [ ] **B3 多情景假设确实不同**：Bull / Base / Bear 三档不能只是数字微调（例如 +10% / 0 / -10%），必须有**质的差异**的假设（如增长率、毛利率、市占率任一项不同）
- [ ] **B4 估值不被单一假设支配**：若敏感性分析显示某一输入（如 WACC、终值倍数、增长率）的小幅变动导致估值大幅摆动，必须在正文点出并附敏感性表
- [ ] **B5 Bull/Bear 不是同一句话的正反面**：例如不允许"Bull：AI 需求爆发 / Bear：AI 需求不爆发"。两边必须各自基于不同维度（如一边讲技术、一边讲监管），且各 ≥3 条

---

## C. 跨市场报告补充（仅 markets 含 ≥2 市场时勾选）

- [ ] **C1 主从关系明确**：从报告 frontmatter `sources` 显式引用主报告路径
- [ ] **C2 §1-§4 引用 + 摘要表**：从报告未冗余复制主报告内容（详见 `references/cross-market-policy.md` §2）
- [ ] **C3 §5/§6/§7 真正差异化**：主报告与从报告在风险、估值、结论上有市场专属内容，不是 copy-paste
- [ ] **C4 AH/ADR 折溢价数据合规**：折溢价数字带 `as_of` 与来源；若 >50% 折价已在文中解释

## 用法说明

1. **报告产出后**，agent 把本 checklist 复制到自检流程中，逐项勾选
2. **任何未通过项**，必须在报告 `## Meta` 中以"未通过项 + 原因"形式说明
3. 严重 sanity 问题（B1-B5 任一 fail 且无合理解释）应**阻断报告产出**，回头补数据或修改结论
4. 自检结果**不需要**写进报告正文，只用于 agent 自身把关

## 报错示例（参考）

- ❌ A3 fail：表格里有"市场份额 6%"但没标 as_of 也没标来源 → 必须补，或挪到"定性描述"
- ❌ B1 fail：净利率 8% 但毛利率只有 6% → 检查是否口径混淆（如毛利不含某些成本），或是否有大额投资收益
- ❌ B5 fail：Bull "国产替代加速" / Bear "国产替代受阻" → 改成 Bull 多维度（产能/客户/技术）、Bear 多维度（地缘/价格战/财务负担）
