# 芯片制造核心概念解读

> 一份写给非半导体专业读者的入门指南
> 更新日期: 2026-05-04

---

## 零、先理解一件事：芯片是怎么造出来的？

在讲具体概念之前，先建立一个全局认知。

芯片制造本质上是在一个 **硅圆盘（晶圆）** 上，用 **光刻机** 像"拍照"一样把电路图案印上去，再用 **刻蚀机** 把不需要的部分"挖掉"，反复几十上百次，最终形成数亿个晶体管组成的集成电路。

```
                      芯片制造简化流程

  沙子 ──→ 硅锭 ──→ 切割成晶圆 ──→ 涂光刻胶 ──→ 光刻(曝光)
    │                                              │
    │                                    ┌─────────┘
    │                                    ▼
    │                              刻蚀(挖掉不要的部分)
    │                                    │
    │                                    ▼
    │                              沉积/掺杂(填入新材料)
    │                                    │
    │                                    ▼
    │                              去胶(洗掉光刻胶)
    │                                    │
    │         ←──────── 重复30-60次 ─────┘
    │                                    │
    │                                    ▼
    │                              测试 ──→ 切割 ──→ 封装 ──→ 成品芯片
    │
    └── 这个过程就是 "晶圆代工 (Foundry)"
```

> **类比理解**：就像在一张A4纸(晶圆)上，用极精细的投影仪(光刻机)投出电路图案，感光涂层(光刻胶)记录下图案，然后用雕刻刀(刻蚀机)沿着图案的线条挖出沟槽，反复几十层，最后在纸上建起一座立体城市。

## 一、晶圆 (Wafer) — 芯片的"地基"

### 1.1 是什么？

晶圆是一块**高纯度的圆形硅片**，是所有芯片制造的基础载体。所有晶体管、导线、绝缘层都在这块硅片上面一层层"长"出来。

### 1.2 怎么来的？

```
  沙子(SiO₂) ──→ 提纯成多晶硅(99.9999999%纯度)
       │
       ▼
  单晶硅生长(拉晶) ──→ 像种水晶一样，从熔融硅中拉出一根圆柱形硅锭
       │
       ▼
  切割成薄片 ──→ 用金刚石线锯切成0.7-0.8mm厚的圆片
       │
       ▼
  研磨抛光 ──→ 表面要像镜子一样光滑(粗糙度<1纳米)
       │
       ▼
  成品晶圆 → 送进Fab(晶圆厂)开始制造
```

### 1.3 关键参数

| 参数 | 主流规格 | 说明 |
|------|---------|------|
| **直径** | 200mm (8英寸) / 300mm (12英寸) | 越大越好→一片晶圆上能切出更多芯片，成本更低 |
| **纯度** | 99.9999999% (9个9) | 每10亿个硅原子中只能有1个杂质 |
| **厚度** | ~0.75mm | 太薄易碎，太厚浪费 |
| **平整度** | <1nm | 比镜面还平整 |

### 1.4 和SMIC的关系

- SMIC目前同时运营 **200mm(8寸)** 和 **300mm(12寸)** 产线
- 8寸用于成熟制程(90nm以上)，成本低，做电源管理、传感器
- 12寸用于先进制程(28nm以下)，做CPU、GPU、AI芯片
- SMIC正在建的四座新厂全部是12寸，目标直指先进制程扩产

### 1.5 为什么越大越难？

12寸晶圆面积是8寸的2.25倍，但制造难度不是线性增加的。更重的晶圆、更严格的平整度要求、更大的设备兼容性挑战。全球能做好12寸的晶圆供应商就那么几家：日本信越(Shin-Etsu)、日本胜高(SUMCO)、德国世创(Siltronic)、中国沪硅产业(NSIG)。

## 二、光刻胶 (Photoresist) — 芯片的"感光底片"

### 2.1 是什么？

光刻胶是一种**对光敏感的化学材料**。涂在晶圆表面后，被光照射到的地方会发生化学反应，变得"可被洗掉"（正胶）或"不可被洗掉"（负胶）。

### 2.2 工作原理

```
  ① 涂胶 ──→ 在晶圆上均匀旋涂一层光刻胶(像涂蛋糕奶油)

  ② 曝光 ──→ 光刻机把电路图案投射到光刻胶上
             被光照到的区域发生化学变化

  ③ 显影 ──→ 用显影液冲洗
             光刻胶上就"显现"出了电路图案

  ④ 刻蚀 ──→ 以光刻胶图形为模板，将图案转移到硅片上

  ⑤ 去胶 ──→ 洗掉光刻胶，留下硅片上的电路图案
```

### 2.3 光刻胶为什么是卡脖子材料？

| 维度 | 详情 |
|------|------|
| **顶级供应商** | JSR(日本)、TOK(日本)、信越化学(日本)、杜邦(美国) |
| **日本垄断** | 全球约90%高端光刻胶由日本企业供应 |
| **国产进展** | 南大光电、晶瑞电材、上海新阳等在攻关，成熟制程已有替代 |
| **禁运风险** | 2019年日韩贸易战中日本曾限制对韩国出口光刻胶 |

> **为什么光刻胶这么难做？** 它不是普通的化学品，而是需要和光刻机的波长、光刻工艺、显影液完美匹配的精密材料体系。换一种光刻胶，整套工艺参数都要重新调，良率会暴跌。这就是为什么晶圆厂一旦选定供应商，极少更换。

## 三、光刻机 — 芯片制造的"心脏"

光刻机是整个半导体制造中最复杂、最昂贵的设备。一台顶级光刻机的价格可以达到 **$2-4亿美元**，包含超过10万个零件。

### 核心理念：为什么光刻机分 DUV 和 EUV？

```
        光刻机的工作原理 = "用光画线"

  光的波长越短 → 能画出的线越细 → 芯片制程越先进

  ┌─────────────────────────────────────────────┐
  │                                              │
  │  DUV (深紫外光)           EUV (极紫外光)      │
  │  波长: 193nm              波长: 13.5nm        │
  │  像一支细笔              像一支极细的笔       │
  │  能画的最小线宽~28-7nm   能画的最小线宽~5-2nm │
  │                                              │
  │  可见光波长~400-700nm (作为参考)              │
  └─────────────────────────────────────────────┘
```

---

### 3.1 DUV (Deep Ultraviolet) 深紫外光刻

#### 是什么？

DUV光刻使用波长 **193nm** 的深紫外光。目前全球绝大多数芯片(28nm以上)都由DUV光刻机制造。

#### 关键技术：浸没式 (Immersion)

传统DUV的问题是光在空气中折射，分辨率受限。浸没式DUV在晶圆和镜头之间注入 **超纯水**，利用水的折射率(1.44)将等效波长压缩到约134nm，大幅提升分辨率。

```
              浸没式DUV光刻示意图

      ┌──────────────┐
      │   激光光源    │ → 产生193nm深紫外光
      └──────┬───────┘
             ▼
      ┌──────────────┐
      │   光罩/掩模   │ → 电路图案的"底片"
      └──────┬───────┘
             ▼
      ┌──────────────┐
      │   投影镜头    │ → 缩小4倍投射到晶圆上
      └──────┬───────┘
             ▼
   ┌─ 超纯水 ────────┐
   │  ════════════   │ ← 关键：水和镜头的界面
   └─ 光刻胶 ────────┘
      ┌──────┴───────┐
      │   硅晶圆      │
      └──────────────┘
```

#### 多重曝光：DUV做先进制程的"偏门"

这是理解SMIC为什么能做7nm/5nm的关键：

```
  台积电/三星用EUV做5nm:
  一次曝光 → 图案刻到晶圆上 → 完成
  (简单、良率高、成本低)

  SMIC用DUV做5nm等效:
  第一次曝光 → 第二次曝光 → 第三次曝光 → 第四次曝光
  → 四次图案叠加 → 才能达到和EUV一次曝光近似的精度
  (这就是SAQP: Self-Aligned Quadruple Patterning)
  
  代价：
  • 曝光次数×4 → 产能÷4 (同样时间只能做1/4的晶圆)
  • 每次曝光都有对准误差 → 良率大幅下降
  • 光刻胶消耗×4 → 材料成本大幅上升
  • 总体效率只相当于EUV的20-30%
```

> **这就是SMIC最大的痛**：不是做不出7nm/5nm，而是用DUV多重曝光做，成本高、良率低、产能浪费。但没办法，EUV被禁运。

---

### 3.2 EUV (Extreme Ultraviolet) 极紫外光刻

#### 是什么？

EUV光刻使用波长仅 **13.5nm** 的极紫外光——这已经接近X射线的范围。EUV是人类有史以来用于工业制造的最短波长光源。

#### 为什么EUV这么难？

| 挑战 | 说明 |
|------|------|
| **光源** | 需要每秒发射5万次激光轰击锡滴，产生等离子体，激发出13.5nm光——功率要250W以上 |
| **反射镜** | 13.5nm光会被空气和玻璃完全吸收！所以不能用普通透镜，必须用钼/硅多层膜反射镜(每面镜子有~100层涂层)，在真空中用反射来聚焦 |
| **反射率** | 每面镜子只能反射约70%，经过11面镜子后只剩约2%的光！所以光源必须极亮 |
| **精度** | 反射镜表面粗糙度必须控制在0.05nm以内——相当于把地球磨成台球那么光滑 |
| **真空** | 整个系统必须在超高真空中运行(因为空气会吸收EUV光) |
| **光罩** | 连光罩(掩模)都必须是反射式的，不能用传统的透射式 |

```
          ASML EUV光刻机(全球唯一的EUV供应商)

         ┌─────────────────────────┐
         │ CO₂激光器(25kW)         │
         │ ↓ 每秒50,000次脉冲      │
         │ 锡滴发生器              │ ← 锡滴直径~30微米
         │ ↓ 激光轰击锡滴→等离子体  │
         │ 13.5nm EUV光源          │
         │ ↓                       │
         │ 收集镜(多层反射)         │
         │ ↓                       │
         │ 照明系统(多面反射镜)     │
         │ ↓                       │
         │ 反射式光罩(电路图案)     │
         │ ↓                       │
         │ 投影镜头(6面反射镜)      │ ← 每面镜子精度<0.05nm
         │ ↓                       │
         │ 晶圆(在真空中)           │
         └─────────────────────────┘

         设备参数：
         • 重量: ~180吨(相当于3辆M1A2坦克)
         • 运输: 需要3架波音747货机分别运送
         • 零件数: >100,000个
         • 单价: $2-4亿美元
         • 年产量: ~50台(ASML产能极限)
         • 耗电量: ~1.5兆瓦(相当于一个小型数据中心的耗电)
```

#### 为什么全球只有ASML能做EUV？

- ASML花了30年、数百亿美元研发EUV
- 整合了全球5000多家供应商的技术(德国蔡司的反射镜、美国Cymer的光源等)
- 关键部件来自美国→所以美国能"长臂管辖"禁止ASML卖EUV给中国
- 台积电是ASML最大客户，三星、英特尔其次
- **中国买不到EUV = 永远无法用最先进方式造芯片**

---

### 3.3 DUV vs EUV 总结对比

| 对比维度 | DUV | EUV |
|---------|-----|-----|
| 光源波长 | 193nm | 13.5nm |
| 供应商 | ASML、尼康、佳能 | 仅ASML一家 |
| 单台价格 | $0.5-1亿 | $2-4亿 |
| 最小制程 | ~7nm(多重曝光) | ~2nm |
| SMIC拥有 | ~160台 | 0台(被禁运) |
| 台积电拥有 | 大量 | 100+台 |
| 透镜类型 | 透射式(玻璃) | 反射式(多层镜) |
| 工作环境 | 空气/水(浸没式) | 真空 |
| 单次曝光效率 | 基准 | ~3-4倍于DUV多重曝光 |

## 四、刻蚀机 (Etcher) — 芯片的"雕刻刀"

### 4.1 是什么？

刻蚀机是**把晶圆上不需要的材料"挖掉"的设备**。光刻胶决定了"哪里要保留，哪里要挖掉"，刻蚀机负责实际的"挖"的动作。

### 4.2 刻蚀在整个流程中的位置

```
  晶圆 ──→ 沉积新材料(像刷墙一样铺一层) ──→ 涂光刻胶
                                                  │
      ┌───────────────────────────────────────────┘
      ▼
  光刻(曝光+显影，光刻胶上形成图案)
      │
      ▼
  刻蚀 ←── 按照光刻胶的图案，把不需要的部分挖掉
      │      (这才是真正在硅片上"雕刻"电路)
      ▼
  去胶(洗掉光刻胶) ──→ 循环重复 ──→ 最终形成多层电路结构
```

### 4.3 两种主要刻蚀方式

#### 湿法刻蚀 (Wet Etching)

- 用化学药液腐蚀
- 便宜、速度快
- 但方向性差(会向侧边腐蚀)→无法做精细结构
- 主要用于大尺寸、非关键层次

#### 干法刻蚀 (Dry Etching) — 这才是核心

- 用等离子体(带电气体)轰击晶圆表面
- 精确控制方向→可以垂直向下挖，几乎不伤侧面
- 是先进制程的核心设备

```
              干法刻蚀原理

       ┌──────────────────────┐
       │   等离子体发生器      │
       │   (CF₄/SF₆/Cl₂等气体) │
       └──────────┬───────────┘
                  ▼
       ⚡⚡⚡⚡ 带电离子+活性自由基 ⚡⚡⚡⚡
                  │
               ┌──┴──┐
               │光刻胶│  ← 保护层
               ├─────┤
               │材料A │  ← 被刻蚀掉
               ├─────┤
               │材料B │  ← 刻蚀停止层
               ├─────┤
               │ 硅  │
               └─────┘

  两种作用机制同时发生：
  ① 物理轰击：离子高速撞击→物理挖掉材料(方向性好)
  ② 化学反应：活性自由基与材料反应→生成挥发性气体被抽走(选择性好)
```

### 4.4 刻蚀机的关键指标

| 指标 | 含义 | 为什么重要 |
|------|------|-----------|
| **刻蚀速率** | 每秒能挖掉多少材料 | 决定产能 |
| **选择性** | 只挖目标材料，不伤光刻胶和下层 | 良率核心 |
| **各向异性** | 垂直向下挖，不向侧面扩张 | 精细度核心 |
| **均匀性** | 整片晶圆上刻蚀深度一致 | 良率核心 |
| **深宽比** | 能挖多深vs多窄的孔/槽 | 3D NAND关键 |

### 4.5 全球刻蚀机格局

| 供应商 | 国家 | 优势领域 |
|--------|------|---------|
| **泛林 Lam Research** | 美国 | 全球刻蚀龙头，介质刻蚀最强 |
| **应用材料 AMAT** | 美国 | 硅刻蚀最强，全产品线 |
| **东京电子 TEL** | 日本 | 介质刻蚀强势 |
| **中微公司 AMEC** | 中国 | 5nm介质刻蚀已进入台积电产线！国产最亮点 |
| **北方华创 NAURA** | 中国 | 硅刻蚀突破中 |

> **重要提示**：刻蚀是中国半导体设备中**国产化进展最好的环节**。中微公司的5nm刻蚀机已经被台积电用于先进制程生产——这意味着在刻蚀这个环节，中国已经部分达到国际先进水平。但离子注入等其他环节的刻蚀设备仍以美国为主。

## 五、这些环节如何串联：一次完整的光刻-刻蚀循环

以制造一个FinFET晶体管为例：

```
  步骤1: 沉积 (在晶圆上铺一层二氧化硅绝缘层)
     │
     ▼
  步骤2: 涂光刻胶 (旋涂，均匀覆盖)
     │
     ▼
  步骤3: DUV/EUV光刻 (曝光+显影)
     → 光刻胶上出现fin图案
     │
     ▼
  步骤4: 刻蚀 (干法，挖掉暴露的二氧化硅)
     → 二氧化硅层上出现fin图案
     │
     ▼
  步骤5: 去胶 (清洗掉光刻胶)
     │
     ▼
  步骤6: 沉积多晶硅 (填入fin凹槽)
     │
     ▼
  步骤7: 涂光刻胶 → 光刻 → 刻蚀 → 去胶
     (形成栅极图案)
     │
     ▼
  步骤8: 离子注入 (向硅中掺入杂质，改变导电性)
     │
     ▼
  ... 重复20-40次类似的循环 ...
     │
     ▼
  完成FinFET晶体管
     │
     ▼
  再重复数十亿次 → 完成整个芯片
```

> **关键认知**：芯片制造不是"一步到位"的，而是**循环往复**。光刻胶→光刻→刻蚀→去胶→再涂胶→再光刻→... 一个先进芯片(如5nm)需要经过约**60-80次**这样的光刻-刻蚀循环。每一次循环的精度都是纳米级(1nm = 十亿分之一米 = 一根头发直径的十万分之一)。

## 六、与SMIC的关联总结

| 概念 | SMIC现状 | 卡脖子程度 |
|------|---------|:----------:|
| **晶圆** | 12寸自供能力弱，依赖进口 | ★★☆☆☆ |
| **光刻胶** | 高端依赖日本，国产在验证 | ★★★☆☆ |
| **DUV光刻机** | 有~160台(ASML)，部分型号受限 | ★★★★☆ |
| **EUV光刻机** | **0台**，完全被禁运 | ★★★★★ |
| **刻蚀机** | 国产替代进展最好(中微/北方华创) | ★★☆☆☆ |

**最后的总结**：

> 芯片制造是人类迄今为止最精密的工业制造过程。光刻机是"画笔"(EUV是极细的笔，DUV是粗一点的笔)，光刻胶是"感光纸"，刻蚀机是"雕刻刀"，晶圆是"画布"。SMIC目前的情况是：画布能买到、雕刻刀有了国产替代、画笔只能用粗的(DUV)靠多画几次来达到细的效果、而极细的笔(EUV)完全买不到——这就是中国芯片自主化最核心的瓶颈。
