## 六、各家优劣势对比

### 华为海思 Ascend — 综合实力断层领先 ✅

- **优势**：唯一拥有自研完整软件栈（CANN + MindSpore + 昇腾云）；华为2025年研发投入超1,600亿¥；已大规模商用验证（华为云、DeepSeek推理、各地智算中心）
- **劣势**：制程受限于SMIC（无法用台积电先进节点）；HBM禁运问题
- **结论**：**最可能杀出重围的一家**

### 海光信息 — 盈利最强的"务实派" ✅

- **优势**：唯一持续盈利的国产AI芯片公司；AMD Zen 1授权 + 自研演进；x86兼容降低生态迁移成本；政府/国企客户粘性极强
- **劣势**：技术底子是AMD 2017年的Zen 1架构，与当前AMD EPYC差距明显；2019年被列入实体清单后无法获取AMD新IP；DCU GPU性能较Nvidia差距大
- **结论**：CPU赛道稳，AI加速器是增量但突破有限

### 寒武纪 — 市值最高、爆发力最强 ⚡

- **优势**：学术背景极强（中科院计算所）；2025年营收爆发式增长520%+；自主指令集架构；2026年首次分红
- **劣势**：曾经最大客户华为2018年自研昇腾后断供；软件生态远不如华为CANN成熟；产品性能仍落后Nvidia同代产品；历史上连续7年亏损
- **结论**：受益于国产替代情绪和强制令，营收爆发，但长期竞争力存疑

### 壁仞科技 — 上限最高、风险最大 ❓

- **优势**：BR100纸面参数惊艳（号称对标H100）；创始团队来自AMD/Nvidia/高通
- **劣势**：受制裁后流片严重受阻；量产前景不确定；软件生态几乎为零
- **结论**：如果没有制裁可能是最强挑战者，但现实处境艰难

## 七、DeepSeek 与国产芯片的关系

```
DeepSeek 推理部署架构：

┌─────────────────────────────────────────┐
│            DeepSeek API 服务              │
├─────────────────────────────────────────┤
│  推理层                                  │
│  ├── 华为昇腾 Ascend 910B/C (主力推理)    │
│  │    └── 制造: SMIC 7nm DUV工艺          │
│  └── Nvidia H800 (部分存量)              │
├─────────────────────────────────────────┤
│  训练层 (DeepSeek-V3/R1 训练)            │
│  └── Nvidia H800 (制裁前囤货)             │
└─────────────────────────────────────────┘
```

关键事实：
- DeepSeek 推理已大规模部署华为昇腾 910B/C，芯片由 **SMIC 7nm工艺**代工
- 这意味着推理环节已打通 **"算法 → 华为芯片设计 → 中芯国际制造"** 纯国产链条
- DeepSeek极低的推理成本，除了算法创新（MoE + MLA），国产芯片性价比也是重要支撑
- **训练端仍严重依赖存量Nvidia显卡**，这一块国产替代还需要时间

## 八、最终结论

| 维度 | 第一名 | 说明 |
|------|-------|------|
| 综合实力 | **华为海思 (Ascend)** | 唯一软硬全栈，唯一大规模商用验证 |
| 盈利能力 | **海光信息 (Hygon)** | 唯一持续盈利，营收规模最大 |
| 增速 | **寒武纪 (Cambricon)** | 2025年营收增长520%，但基数从很低开始 |
| 国产替代紧迫性 | **SMIC + 华为** | 制造+设计的唯一闭环组合 |

**谁最可能杀出重围？**

短期（1-3年）：**华为昇腾** 是无争议的答案——软件生态、客户部署量、政府支持力度、与SMIC的深度绑定，各方面都是降维打击。

长期风险：整个中国AI芯片赛道都面临同样的天花板——SMIC制程落后台积电2-3代、HBM禁运、EDA工具受限。这决定了国产AI芯片的"能力上限"。但推理端（对制程要求相对低）已经证明可以做到"够用且便宜"，DeepSeek就是最佳例证。

#### SMIC vs 其他代工厂 — 代差有多大？

```
制程代际对比:

SMIC:    N+3(5nm等效) → N+2(7nm) → 14nm → 28nm → 40nm → ...
华虹:                              28nm → 40nm → 55nm → ...
晶合:                                     55nm → 90nm → 110nm → ...
芯联:             (限90nm以上成熟制程)
和舰:                            (110nm以上成熟制程)
```

**关键结论：SMIC 和第二名华虹之间，差了整整 4-5 个制程节点。这不是同一梯队的竞争。**

### 9.2 各代工厂详细分析

#### 9.2.1 中芯国际 SMIC — 毫无疑问的国产第一

| 维度 | 详情 |
|------|------|
| **先进制程** | 14nm FinFET(2019量产) → N+2/7nm(2022量产) → N+3/5nm等效(2025突破) |
| **产线布局** | 上海(S1/SN1/S2/临港)、北京(B1/B2/京城)、深圳(Fab15+新厂)、天津(Fab7+西青)、意大利(LFoundry) |
| **设备来源** | ASML DUV浸没式光刻机(制裁前大量囤货，约160台DUV设备)，以及应用材料、泛林等美国设备(部分受限) |
| **核心客户** | 华为海思(Ascend AI芯片)、高通、博通、TI、国内Fabless |
| **扩张中产能** | 京城(北京100K/月)、临港(上海100K/月)、深圳(40K/月)、西青(天津100K/月) |
| **制裁影响** | EUV被完全禁运，DUV部分受限(NXT:2050i以上型号需许可)，无法获得台积电级别的EUV先进制程 |

**SMIC的"独一档"地位体现在：**
- 国产AI芯片（华为Ascend、寒武纪思元）的唯一可量产代工厂
- 中国唯一能制造7nm以下逻辑芯片的工厂
- 华为Mate 60（麒麟9000S，SMIC N+2）、Mate 80（麒麟9030，SMIC N+3）都依赖SMIC
- DeepSeek推理用的华为昇腾910B/C芯片均由SMIC代工

**局限性：**
- N+3(5nm等效)使用的是DUV多重曝光（SAQP技术），良率和成本远不如台积电EUV 5nm
- 与台积电N3/N2至少差2-3代，且差距在被EUV禁运的情况下难以缩小
- 2026年4月美国又对华虹实施设备禁令，SMIC的设备获取也持续受限

#### 9.2.2 华虹半导体 — 老二，但差距巨大

| 维度 | 详情 |
|------|------|
| **最先进制程** | 28nm/22nm（子公司上海华力HLMC） |
| **特色工艺** | BCD(功率管理)、eNVM(嵌入式存储)、CIS(图像传感器)、IGBT |
| **产线** | 3座200mm厂(月产18万片) + 2座300mm厂(华力) + 无锡新厂(HH Fab7在建) |
| **定位** | 特色工艺代工，不做先进逻辑，对标台积电的成熟制程+特色工艺路线 |
| **制裁** | 2026年4月美国商务部要求设备商停止向华虹供货部分设备 |

华虹的战略是不和SMIC在先进制程上正面竞争，而是深耕模拟/功率/嵌入式存储等"足够用的"特色工艺。这也更符合中国大多数芯片设计公司的需求。

#### 9.2.3 晶合集成 Nexchip — 老三，专注显示驱动

| 维度 | 详情 |
|------|------|
| **最先进制程** | 55nm（研发中），量产主力90nm |
| **核心产品** | 显示驱动IC(DDI)占营收87.84%，其次是CIS和PMIC |
| **背景** | 台湾力晶(Powerchip)与合肥市政府合资 |
| **产能** | 12寸晶圆，月产约5万片 |

晶合基本不做AI芯片代工，属于半导体供应链中游的"基建型"代工厂。

#### 9.2.4 特殊定位的玩家

| 公司 | 核心方向 | 全球竞争力 |
|------|---------|-----------|
| **长江存储 YMTC** | 3D NAND闪存，232层堆叠 | 全球领先水平，已超越铠侠/西数 |
| **长鑫存储 CXMT** | DRAM，19nm→17nm | 追赶中，与三星/海力士差距约1-2代 |

### 9.4 "国产最好的就是中芯国际" — 结论

**是的，而且不是"相对好一些"，是断层式领先。**

| 对比维度 | SMIC | 其他国产代工厂 |
|---------|------|--------------|
| 最先进制程 | 7nm/5nm等效 | 28nm(华虹) |
| 制程差距 | — | 落后SMIC 4-5代 |
| 可做AI芯片？ | Yes (华为Ascend) | No |
| 可做先进手机SoC？ | Yes (麒麟9000S/9030) | No |
| 可做先进CPU？ | 有挑战但能做 | No |
| 全球排名 | 第3 | 华虹第6，晶合第10左右 |
| 市值 | ~6,088亿¥ | 华虹 ~600亿¥，晶合 ~300亿¥ |

**但核心痛点不是SMIC本身的能力，而是它赖以运行的设备：**

- SMIC的7nm是用 **ASML DUV + 多重曝光(SAQP)** 实现的，这是一种"降级方式"做先进制程
- 台积电用EUV一次曝光就搞定的事情，SMIC需要四次曝光，导致良率低、成本高
- 最关键的光刻机来自荷兰ASML(EUV被禁运，DUV受限)、刻蚀/薄膜设备大量依赖美国(应用材料/泛林)
- **国产设备替代(北方华创、中微公司、上海微电子)在先进制程上差距巨大**

> **一句话总结：SMIC是中国芯片自主化的"单点支撑"，在禁运条件下用二流设备做出了一流工艺，已经很了不起了。但整个供应链的自主化(特别是EUV光刻机)才是最终解。**
