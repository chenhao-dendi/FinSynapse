## 一、DeepSeek V4 基本情况

| 项目 | V4-Pro | V4-Flash |
|------|--------|----------|
| **发布日期** | 2026-04-24 (Preview) | 2026-04-24 (Preview) |
| **总参数** | 1.6T (1.6万亿) | 284B (2840亿) |
| **激活参数** | 49B | 13B |
| **上下文窗口** | 1M tokens | 1M tokens |
| **架构** | MoE + Hybrid Attention | MoE + Hybrid Attention |
| **许可证** | MIT | MIT |
| **公司** | 杭州深度求索 (DeepSeek) | 同 |
| **母公司** | 幻方量化 (High-Flyer) | 同 |
| **CEO** | 梁文锋 | 同 |
| **总部** | 杭州, 浙江 | 同 |
| **最新估值** | ~$100亿 (2026年4月融资轮) | 同 |

**核心技术架构创新:**
- **Hybrid Attention**: CSA (Compressed Sparse Attention) + DSA (DeepSeek Sparse Attention) + HCA (Heavily Compressed Attention)
- **Engram**: 记忆-计算分离架构，静态知识存 CPU RAM，动态推理走 GPU
- **mHC**: 流形约束超连接 (Manifold-Constrained Hyper-Connections)，解决超大规模 MoE 训练不稳定性
- **DSA**: 稀疏注意力，近线性 scaling，60-70% 内存带宽优化
- **KV Cache 减少 90%**，单 token 推理 FLOPs 减少 73%（对比 V3.2）

## 二、芯片供应链

### 2.1 训练芯片

| 层级 | 厂商/产品 | 状态 | 说明 |
|------|----------|------|------|
| **传闻训练芯片** | NVIDIA Blackwell (B200/GB200) | ⚠️ 争议 | 美方官员声称 V4 使用走私的 Blackwell 芯片训练；DeepSeek V4 技术报告对此保持沉默（V3 报告明写了 A100/H800） |
| **已确认推理芯片** | 华为 Ascend 950 系列 (910B/950PR) | ✅ 确认 | V4 是首个完整针对华为 Ascend 芯片做原生优化的 Tier-1 模型 |
| **历史训练芯片** | NVIDIA A100 / H800 | ✅ 历史使用 | V2/V3/R1 时代使用，受美国出口管制影响 |
| **V4-Flash 训练** | 华为 Ascend 芯片 | ✅ 确认 | 华为确认 Ascend 芯片参与了 V4-Flash 的训练 |

### 2.2 推理芯片（部署端）

| 层级 | 厂商/产品 | 定位 | 详情 |
|------|----------|------|------|
| **主推理芯片 (中国)** | 华为 Ascend 950PR | 国产首选 | 昇腾 950PR，2026年3月开始量产；华为预计2026年出货 ~75万片 |
| **下一代芯片 (中国)** | 华为 Ascend 950DT | 升级版 | 计划2026年Q4发布 |
| **推理芯片 (海外)** | NVIDIA Blackwell B300 | 国际部署 | NVIDIA build.nvidia.com 提供 V4 推理端点 |
| **推理芯片 (海外)** | NVIDIA GB200 NVL72 | 高吞吐 | 实测 DeepSeek-V4-Pro >150 tokens/sec/user |
| **此前推理芯片** | NVIDIA A100 / H800 | 旧平台 | V3/R1 时代主力推理硬件 |

### 2.3 GPU 利用率对比（关键指标）

| 平台 | 此前天花板 | V4 达成 | 提升幅度 |
|------|-----------|---------|---------|
| **NVIDIA CUDA** | ~90%+ | ~90%+ | 成熟生态 |
| **华为 Ascend (CANN)** | ~60% | **85%+** | +25pp，约 40% 硬件成本降低 |

> DeepSeek 与华为联合做 Kernel 级优化（CANN 框架），针对 MoE routing、sparse attention、Engram memory retrieval 进行定制化适配。

### 2.4 芯片制造环节

| 环节 | 厂商 | 说明 |
|------|------|------|
| **Ascend 芯片代工** | 中芯国际 (SMIC) | 华为 Ascend 处理器制造商；V4 发布后 SMIC 港股单日涨 10% |
| **Ascend 芯片设计** | 华为海思 | 自研达芬奇架构 |
| **先进制程设备** | ASML / 应用材料 / 东京电子 / Lam Research | 受美国出口管制，限制中国获取 EUV 等设备，制约 Ascend 扩产 |
| **NVIDIA GPU 制造** | TSMC (台积电) | Blackwell/Hopper 系列代工 |
| **HBM 内存** | SK 海力士 / 三星 / 美光 | GPU 配套高带宽内存，也受出口管制影响 |

## 八、关键结论摘要

1. **DeepSeek V4 是首个完整针对华为 Ascend 做原生优化的 Tier-1 模型**，这标志着中国 AI 产业链正在脱离 NVIDIA 生态。
2. **训练芯片来源存在争议**：美方声称使用走私 Blackwell，DeepSeek 技术报告对此沉默。
3. **华为是最大商业赢家**：Ascend 950 订单暴增，2026 年 AI 芯片收入目标 $120亿 (+60%)。
4. **SMIC 是关键制造瓶颈**：产能受美国设备出口管制限制。
5. **定价颠覆性**：V4-Flash 比 GPT-4 Turbo 便宜 ~107倍，可能迫使西方厂商降价。
6. **全球 AI 基础设施正在分裂为两个平行体系**：西方(NVIDIA+CUDA) vs 中国(Ascend+CANN)。
7. **蒸馏攻击指控**可能触发进一步制裁，是最大政策风险点。
8. **DeepSeek 自身面临芯片短缺**，无法大规模服务 V4-Pro。
