# 泡泡玛特 / Pop Mart International (9992.HK) 分析报告

## TL;DR

泡泡玛特是中国最大的潮玩 IP 运营商（盲盒收藏品 + 自有/授权 IP 矩阵 + 直营/机器人商店/海外 全渠道），2020-12 港股上市。FY2024 营收约 ¥130 亿（YoY +100%+，**估算**），核心驱动是 **Labubu (THE MONSTERS)** 在 2024-2025 全球爆火（泰国 / 东南亚 / 北美），海外营收占比从 FY2023 的 ~17% 跃升至 FY2024 ~30-40%（**估算**）。
**当下叙事**：从"中国盲盒龙头"重定位为"全球潮玩 IP 平台"，估值锚被市场拉到与全球玩具/IP 巨头（孩之宝、万代南梦宫）对标。
**多空核心**：Bull——Labubu 全球破圈印证 IP 出海方法论 + 高毛利（60%+）+ 海外门店单店产出远超国内；Bear——单 IP 过度依赖（Labubu 占增量营收估算 50%+）+ 盲盒商业模式可持续性争议 + 估值已 price in 海外多年增长。
**研究结论**：**中性**——海外破圈逻辑成立但单 IP 集中度极高，估值消化需要 Labubu 不减温 + 第二曲线（如 Crybaby / Hirono）跑出。详见第 7 章。

## 1. 业务画像（What it does）

### 1.1 商业模式

**潮玩 IP 收藏品零售**——核心是 IP 内容运营 + 实体商品销售，结构上类似"消费版迪士尼/万代"但规模与 IP 深度差距巨大。

四个收入环节：
1. **IP 设计 / 收购**：自有 IP（Molly 等）+ 授权 IP（Labubu / Crybaby 等）+ 与艺术家分成
2. **产品开发**：盲盒、手办、毛绒、积木、玩具
3. **零售渠道**：直营店 + 机器人商店 + 抽盒机 + 电商 + 海外门店 + 第三方批发
4. **会员与运营**：会员体系、二手交易（葩趣 APP）

### 1.3 客户与渠道（含集中度）

- **客户画像**：18-35 岁女性为主（占比 ~75%），一二线城市中高收入；海外用户画像因地域差异大（东南亚以年轻女性为主，北美包含潮流/收藏圈层）
- **客户集中度**：极度分散（C 端零售），无单一大客户依赖（与 SMIC 截然不同的优势）
- **渠道集中度**：自营渠道占 70%+，对单一第三方平台依赖低

### 1.4 行业地位与市场份额

- **中国潮玩市场**：第 1 名，远超第二名（52TOYS、TOP TOY 等）；市场份额估算 ~20-25%（中国潮玩市场 ~600-800 亿规模）
- **全球潮玩市场**：开始进入全球前列玩家行列，但与孩之宝（HAS）、万代南梦宫（NCBDY）等综合玩具巨头规模差距明显
- **核心壁垒**：IP 矩阵 + 全球渠道运营能力（特别是机器人商店与抽盒机这种独特模式）

## 3. 护城河与竞争（Why it wins/loses）

### 3.1 核心竞争优势

| 优势 | 说明 | 类别 |
|------|------|------|
| IP 矩阵丰富 | 自有 + 授权 + 艺术家分成 三种模式覆盖头中尾 IP，单 IP 风险有缓冲 | IP / 内容 |
| 全球渠道运营经验 | 直营店 / 机器人商店 / 抽盒机 / 海外门店 / 电商，**机器人商店与抽盒机几乎是中国独有的高效零售形态** | 渠道 / 运营 |
| 全球化扩张方法论 | 泰国（Crybaby 本土化）+ 北美（Labubu 名人带动）的成功路径已被验证 | 全球化 |
| 高毛利 + 高现金流 | 与 SMIC 相反——可以投入到新 IP 孵化、海外扩张、艺术家收购 | 财务结构 |
| 二手与社区运营 | 葩趣 APP / 抽盒机 / 二手交易让 IP 价值在销售后继续增长 | 用户粘性 |

### 3.3 替代品与新进入者威胁

- **新进入者**：低门槛（盲盒形态易复制），但 **IP 矩阵 + 渠道网络是真正护城河**——TOP TOY 等追赶者多年未能形成对标规模
- **替代品**：游戏内购、虚拟收藏品（NFT）、二次元周边、汉服等都在抢 Z 世代时间与钱包
- **国际竞争**：进入海外后直接面对孩之宝 / 万代等本土巨头，但目前 Pop Mart 主打"中国设计美学 + 盲盒形态"形成差异化

## 5. 增长驱动与风险（Where it goes）

### 5.1 看多逻辑（Bull case）

1. **Labubu 全球破圈印证 IP 出海方法论**（IP 维度）：BLACKPINK Lisa 等带动后，Labubu 在泰国、菲律宾、北美形成现象级文化符号——这种破圈一旦达成，回收周期是多年的
2. **海外门店单店产出远超国内**（运营效率维度）：曼谷、新加坡、洛杉矶等核心城市的旗舰店年化营收估计是国内一线城市的 2-4 倍，且毛利率更高
3. **第二曲线 IP 储备充足**（IP 储备维度）：Crybaby（泰国本土）、Hirono、Dimoo 等若任意一个跑出"次 Labubu"，营收上限大幅打开
4. **轻资产 + 高现金流支持持续投入**（财务结构维度）：年 FCF 约 ¥35-45 亿（估算）支持每年 100+ 家海外新店扩张
5. **机器人商店与抽盒机模式可复制到海外**（渠道创新维度）：这种零售形态在海外几乎没有竞争对手，先发优势明显

### 5.2 看空逻辑（Bear case）

1. **单 IP 集中度过高**（IP 风险维度）：Labubu 占增量营收估算 50%+，若热度退潮（潮玩历史平均生命周期 3-5 年），下行风险大
2. **盲盒商业模式面临监管压力**（监管维度）：中国 2023 年起加强未成年盲盒销售监管，未来可能扩展到价格、抽数、信息披露等
3. **消费降级周期**（宏观维度）：潮玩属可选消费，若中国 / 全球进入消费下行，单价 ~¥69-99 的盲盒首当其冲
4. **海外扩张成本曲线尚未充分验证**（扩张风险维度）：当前海外门店爬坡数据来自最热的一批（曼谷、新加坡），若进入次级城市，单店产出可能大幅下降
5. **估值已 price in 多年高增长**（估值维度）：当前 PE 估算 35-50 倍（**估算**），隐含 3-5 年净利润 CAGR 30%+，留给意外的空间小
6. **文化适应风险**（全球化维度）：中国潮玩在东南亚 / 欧美的成功有 BLACKPINK 等"明星带货"偶发性，能否形成稳态仍需观察

### 5.3 监管 / 地缘 / 合规

| 风险类型 | 状态 | 影响 |
|---------|------|------|
| 中国盲盒监管 | 2023 起持续 | 未成年限购、信息披露要求；进一步扩展可能影响销售模式 |
| 海外消费品法规 | 各国不同 | 玩具安全、税务、儿童产品标识；东南亚相对宽松，欧美更严 |

### 7.1 研究结论

**中性**

### 7.2 当前 Thesis（一句话）

泡泡玛特已从"中国盲盒龙头"成功重定位为"全球 IP 平台候选"，**Labubu 全球破圈印证了 IP 出海方法论**，但**单 IP 集中度过高 + 估值已 price in 多年高增长**，研究上需要等待**第二曲线 IP 跑出**或**Labubu 显示出超预期持续性**才能更明确——当前观察。

### 7.3 强化事件（thesis 更成立）

1. **2026 Q2-Q3 业绩**：FY2025 全年营收 ≥ ¥250 亿且海外占比 ≥ 45%
2. **第二爆款 IP**：Crybaby 或 Hirono 在海外某市场（不限于亚洲）月销售跻身 IP Top 3
3. **北美突破**：北美门店数 ≥ 50 且单店年化营收 ≥ ¥1500 万
4. **Labubu 持续性证明**：进入 2026 年下半年 Labubu 月销同比仍在 +30%+
5. **欧洲市场启动**：伦敦 / 巴黎 / 柏林等核心城市旗舰店开业且首月数据强劲

### 7.4 削弱事件（thesis 受损）

1. **Labubu 月销环比连续 3 个月下滑** > 10%
2. **第二曲线缺位**：FY2026 H1 时 Labubu 仍占增量营收 50%+
3. **海外开店速度明显放缓**（季度净增 < 10 家）
4. **创始人健康 / 重大事件**（key man risk 触发）
5. **中国出台具体盲盒定价或抽数管制**

### 7.5 证伪事件（推翻 thesis）

1. **Labubu IP 授权出现重大分歧或终止**（公司与 Kasing Lung 的合作关系破裂）
2. **重大食品/玩具安全事件**导致海外被禁
3. **海外某主要市场（如美国）出台针对中国玩具的关税或贸易限制**
