## 二、业务发展

### 2.1 业务定位与行业地位

| 维度 | 详情 |
|------|------|
| **商业模式** | Pure-play晶圆代工（只做制造，不设计芯片） |
| **全球排名** | 第3大晶圆代工厂（2024年，仅次于台积电、三星） |
| **中国排名** | 第1大晶圆代工厂（大陆唯一先进制程代工厂） |
| **市场份额** | 全球约6%（台积电约62%，三星约12%） |
| **总部/注册地** | 上海运营 / 开曼群岛注册 |
| **员工人数** | ~20,000人 |

### 3.2 关键技术参数对比

| 指标 | SMIC N+2 (7nm) | SMIC N+3 (5nm等效) | 台积电 N5 (5nm) | 差距 |
|------|---------------|-------------------|----------------|------|
| 实现方式 | DUV + 多重曝光 | DUV + SAQP(自对准四重图案) | EUV + 单次曝光 | 技术路径不同 |
| 晶体管密度 | ~90 MTr/mm² | ~130 MTr/mm²(估算) | ~170 MTr/mm² | 约25-30% |

### 3.3 核心技术难点与突破方式

| 技术瓶颈 | 国际方案 | SMIC方案 | 优劣 |
|---------|---------|---------|------|
| **EUV光刻** | ASML EUV(13.5nm波长) | ❌ 被禁运 | 最大卡脖子点 |
| **DUV替代方案** | — | ASML DUV浸没式(193nm波长) + SAQP四重曝光 | ✅ 绕过禁运，但良率/成本劣化 |
| **先进光刻胶** | JSR/TOK(日本) | 国产替代(南大光电等) | 部分可用，差距仍在 |
| **EDA工具** | Synopsys/Cadence | 华大九天 + 自研 | 成熟制程可用，先进制程受限 |
| **HBM内存** | SK海力士/三星 | 无法获取 | 影响AI芯片性能上限 |

### 3.4 SMIC的设备"粮仓"

> 据公开报道，SMIC在美国2020年制裁前大量囤积半导体设备。

| 设备类型 | 估计保有量 | 关键供应商 | 制裁状态 |
|---------|-----------|-----------|---------|
| DUV光刻机 | ~160台(ASML) | ASML(荷兰) | 部分受限(NXT:2050i以上) |
| 刻蚀机 | 数百台 | Lam/AMAT(美) + 中微公司 | 美国设备受限 |

> **SMIC的设备策略**：制裁前大量囤货 + 国产替代逐步切换 + 二手设备补充。据估算囤货可支撑2-3年扩张，之后必须依赖国产设备突破。

## 五、未来的最大发展潜力

### 5.1 潜力一：AI推理芯片代工的"独家生意"

```
        中国AI推理芯片制造格局

  华为海思 Ascend ────→ 需要7nm/5nm制造
  寒武纪 思元 ────────→ 需要7nm制造          ┐
  壁仞科技 BR系列 ────→ 需要7nm制造          │
  燧原科技 云燧系列 ──→ 需要7nm制造          ├── 全部只能找SMIC
  摩尔线程 MTT系列 ──→ 需要先进制程          │
  其他AI芯片公司 ─────→ 需要先进制程          ┘

  中国大陆能做的 ────→ 只有SMIC一家
```

**这是SMIC最大的结构性优势**：在中国AI芯片需求爆发的背景下，所有国产AI芯片设计公司的先进制程制造需求，**唯一可选的就是SMIC**。这种"垄断性需求"保证了未来5-10年的订单增长。

### 5.2 潜力二：产能大扩张的量价双击

| 新增工厂 | 规划产能 | 投资额 | 预计投产 |
|---------|---------|--------|---------|
| 京城(北京) | 10万片/月(12寸) | $77亿 | 2025-2026 |
| 临港(上海) | 10万片/月(12寸) | $88.7亿 | 2025-2026 |
| 深圳新厂 | 4万片/月(12寸) | $23.5亿 | 2025-2027 |
| 西青(天津) | 10万片/月(12寸) | $75亿 | 2026-2028 |

> **四大新厂合计新增34万片/月12寸产能，几乎等于再造一个SMIC。**

### 5.3 潜力三：国家大基金持续输血

| 投资方 | 持股比例(约) | 备注 |
|--------|------------|------|
| 国家集成电路产业投资基金(大基金) | ~23% | 中国芯片产业的"国家队" |
| 上海集成电路产业投资基金 | ~11.5% | 上海市级支持 |
| 大唐电信 | ~12% | 央企股东 |
| 2020年注资 | 合计$20亿 | 大基金+上海基金联合注资 |

> **有国家队加持，SMIC的资本开支几乎不受盈利压力的约束。$73亿/年的烧钱速度，普通企业早撑不住了，但SMIC可以。**

### 5.4 潜力四：从"能用"到"好用"的技术跃迁

- N+2(7nm)已经成熟→良率爬坡后利润率将改善
- N+3(5nm等效)已突破→2025-2026年良率提升关键期
- 如果未来中美关系缓和，或国产EUV实现突破→SMIC的制程天花板将大幅提升
- 即使不突破EUV，DUV+SAQP路线也能走到~3nm等效(理论极限)

### 5.5 远景测算

```
情景一(乐观): AI芯片需求持续爆发 + 四大新厂满产 + N+3良率突破
  → 年营收可达$200亿+(当前$93亿, 约115%增长空间)
  → 净利润率有望回升至15-20%

情景二(中性): 成熟制程稳定增长 + 先进制程逐步放量
  → 年营收$120-150亿，维持全球第3-4位

情景三(悲观): 美国进一步收紧设备禁令 + 新厂建设受阻
  → 增长停滞，年营收维持$80-100亿
```

```
  │   │  • 三星有50+台                    │     │
  │   │  • SMIC有0台 ← 被禁运             │     │
  │   │  • 单台价格: $2-4亿               │     │
  │   └──────────────────────────────────┘     │
  │                                            │
  │   SMIC只能使用DUV替代:                      │
  │   ┌──────────────────────────────────┐     │
  │   │  DUV浸没式(193nm波长)             │     │
  │   │  + 多重曝光(SAQP)                 │     │
  │   │  = 用4次曝光模拟1次EUV的效果      │     │
  │   │  → 代价: 良率低、成本高、产能浪费 │     │
  │   └──────────────────────────────────┘     │
  └────────────────────────────────────────────┘
```

**这是SMIC最根本的制约。** 没有EUV，就无法做出真正对标台积电的先进制程。DUV多重曝光是一条"偏门"，能走到5nm等效已是极限，再往下(3nm、2nm)几乎不可能。

### 6.2 制约二：与台积电的制程代差在拉大

| 时间 | 台积电 | SMIC | 代差 |
|------|--------|------|------|
| 2019 | 7nm(EUV)量产 | 14nm(DUV)量产 | ≈2代 |
| 2020 | 5nm(EUV)量产 | 14nm扩产 | ≈3代 |
| 2022 | 3nm(EUV)量产 | N+2/7nm(DUV)量产 | ≈2代 |
| 2024 | 3nm成熟 + 2nm试产 | N+3/5nm等效(DUV) | ≈2-3代 |
| 2025-2026 | 2nm(GAA)量产 | N+3扩产 | ≈3代+ |

> **坏消息**：SMIC虽然突破了7nm和5nm等效，但台积电也在前进。代差维持在2-3代，短期内难以缩小。
> **好消息**：对于AI推理等大规模部署场景，7nm/5nm等效的芯片已经"够用"，不需要最顶尖的制程。

### 6.3 制约三：设备禁令随时可能升级

| 已发生的禁令 | 时间 | 影响 |
|-------------|------|------|
| 实体清单(Entity List) | 2020.12 | 美国技术/设备出口SMIC需许可 |
| 军事终端用户清单 | 2020.09 | 美国投资禁令 |
| EUV禁运(荷兰配合) | 2019-今 | 最先进光刻机完全无法获取 |
| DUV部分限制 | 2023+ | NXT:2050i及以上型号受限 |
| 华虹设备禁令(最新) | 2026.04 | 美国扩大对中国代工厂设备禁令 |

> **最大的风险**：如果美国进一步收紧到"任何ASML DUV设备都需要许可"，SMIC将面临产能扩张停滞。目前囤的设备约能支撑2-3年。

### 6.4 制约四：财务压力

| 财务压力点 | 详情 |
|-----------|------|
| **巨量资本开支** | $73亿/年(CapEx)，远超经营现金流 |
| **高折旧压力** | 年折旧约$22亿，且随着新厂投产将继续增加 |

| **毛利率低** | 18-22%，远低于台积电(53%+)，甚至低于联电(35%+) |

### 6.5 制约五：人才与外部环境

## 七、综合评价

| 维度 | 评分 | 说明 |
|------|:----:|------|
| 战略地位 | ★★★★★ | 中国芯片自主化的"命门"，无可替代 |
| 技术实力 | ★★★★☆ | DUV做出5nm等效是全球唯一，但距台积电仍有代差 |
| 财务健康 | ★★★☆☆ | 营收增长但利润率低、烧钱严重 |
| 成长潜力 | ★★★★★ | AI推理需求+产能大扩张+国产替代，三重驱动 |
| 抗风险能力 | ★★☆☆☆ | 严重依赖进口设备，禁令升级风险极高 |

**一句话结论**：SMIC是**中国半导体自主化最不可或缺的一环，也是最脆弱的一环**。它的战略价值远超其财务价值——这就是为什么市场给它188倍PE。成功与否，最终取决于三点：(1)美国禁令会不会进一步升级，(2)ASML DUV囤货能用多久，(3)国产设备(特别是光刻机)何时能顶上。
